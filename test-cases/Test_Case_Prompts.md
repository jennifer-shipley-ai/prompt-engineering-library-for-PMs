# Test Case Prompts for PMs

Prompts to generate comprehensive test cases, edge scenarios, and validation steps.

---

## 1. Generate Test Cases from User Stories
**Prompt:**  
“Generate detailed test cases for this user story. Include positive, negative, and edge scenarios.”

---

## 2. Acceptance Criteria → Test Cases
**Prompt:**  
“Convert these acceptance criteria into executable test cases with clear steps, expected results, and data inputs.”

---

## 3. Regression Test Identification
**Prompt:**  
“Identify regression test cases that should be added based on this feature’s impact on existing workflows.”

---

## 4. Risk-Based Testing
**Prompt:**  
“Analyze this feature and generate test cases focused on high-risk areas such as data integrity, security, and compliance.”

---

## 5. UAT Scenario Generation
**Prompt:**  
“Generate UAT scenarios that reflect real-world usage patterns for this feature. Include business validation steps.”

---

These prompts help PMs ensure quality, reduce defects, and accelerate release readiness.
