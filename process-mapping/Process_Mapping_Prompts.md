# Process Mapping Prompts for PMs

Prompts to accelerate workflow discovery, visualization, and optimization.

---

## 1. Generate Current-State Process Maps
**Prompt:**  
“Describe the current-state workflow for <process>. Include actors, systems, decision points, bottlenecks, and handoffs.”

---

## 2. Identify Process Inefficiencies
**Prompt:**  
“Analyze this workflow and identify inefficiencies, delays, manual steps, and automation opportunities.”

---

## 3. Create Future-State Process Maps
**Prompt:**  
“Design a future-state workflow that incorporates AI automation. Highlight improvements, removed steps, and new roles.”

---

## 4. Swimlane Diagram Description
**Prompt:**  
“Generate a swimlane-style description of this workflow, organized by role or system.”

---

## 5. Process Risk Assessment
**Prompt:**  
“Identify risks in this workflow related to data, compliance, user error, and system dependencies. Suggest mitigations.”

---

These prompts help PMs map, optimize, and modernize processes with clarity.
