# Prompt Engineering Library for PMs

A practical, PM-focused library of prompts designed to accelerate project delivery using AI tools.  
This collection demonstrates how project managers can use prompt engineering to improve clarity, speed, and alignment across requirements, user stories, testing, process mapping, and portfolio reporting.

---

## 📁 What’s Inside

### Requirements Prompts
Prompts for eliciting, clarifying, prioritizing, and validating business and technical requirements.

### User Story Prompts
Prompts for generating user stories, improving quality, and creating acceptance criteria.

### Test Case Prompts
Prompts for generating test cases, regression scenarios, UAT flows, and risk-based testing.

### Process Mapping Prompts
Prompts for documenting current-state workflows, designing future-state processes, and identifying automation opportunities.

### Portfolio Reporting Prompts
Prompts for creating executive-ready status summaries, KPI narratives, risk reports, and portfolio roll-ups.
